# Rule of Bool: Eternity Relative Time

```python
def Robert():
    X = 0
    Y = 0
    while True:
        if (True or False):
            X => X += 1
        else:
            X => Y += 1
```

```CSharp
Protected IEnumerable<bool> Robert()
{
    while (true) X++ else Y++;
}
```
